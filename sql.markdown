## SQL ISO/IEC 9075

IsentaDB uses a Structured Query Language that closely follows the ISO/IEC 9075 standard, allowing developers to write familiar SQL with minimal adaptation.  

In addition to the standardized SQL features, IsentaDB provides its own extensions that are not part of the ISO/IEC 9075 specification. These database-specific features are documented in the section Beyond SQL.  

Every query and keyword is fully documented and can be accessed at any time using the `help` command in the IsentaDB Shell.
